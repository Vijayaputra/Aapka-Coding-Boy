HTML LIST

LIST

HTML lists allow web developers to group a set of related items in lists.

Example
  • <!DOCTYPE html>
  • <html>
  • <body>

  • <h2>An Unordered HTML List</h2>

  • <ul>
  • <li>Coffee</li>
  • <li>Tea</li>
  • <li>Milk</li>
  • </ul>

  • <h2>An Ordered HTML List</h2>

  • <ol>
  • <li>Coffee</li>
  • <li>Tea</li>
  • <li>Milk</li>
  • </ol>

  • </body>
  • </html>

OUTPUT

An Unordered HTML List

  • Coffee
  • Tea
  • Milk

An Ordered HTML List

  1. Coffee
  2. Tea
  3. Milk

ORDER LIST

  • An ordered list starts with the <ol> tag. Each list item starts with the <li> tag.
  • The list items will be marked with numbers by default:
  • TYPE-
    1.type="1" (default)
    2.type="A"
    3.type="a"
    4.type="I"
    5.type="i"
  • Attributes-
    1.start eg.start="10"
    2.reversed (optional)
    3.type
Example 1
  • <!DOCTYPE html>
  • <html>
  • <body>

  • <h2>An Ordered HTML List</h2>

  • <ol start="5">
  • <li>Coffee</li>
  • <li>Tea</li>
  • <li>Milk</li>
  • </ol>

  • </body>
  • </html>

OUTPUT

An Ordered HTML List

  1. Coffee
  2. Tea
  3. Milk

UNORDER LIST

  • An unordered list starts with the <ul> tag. Each list item starts with the <li> tag.
  • The list items will be marked with bullets (small black circles) by default:
  • TYPE-
    1.type="disc" (default)
    2.type="square"
    3.type="circle"
    4.type="none"
Example 1
  • <!DOCTYPE html>
  • <html>
  • <body>

  • <h2>An Unordered HTML List</h2>

  • <ul>
  • <li>Coffee</li>
  • <li>Tea</li>
  • <li>Milk</li>
  • </ul>

  • </body>
  • </html>

DEFINATION LIST

  • An unordered list starts with the <ul> tag. Each list item starts with the <li> tag.HTML also supports defination lists.
  • A defination list is a list of terms, with a defination of each term.
  • The <dl> tag defines the defination list, the <dt> tag defines the term (name), and the <dd> tag describes each term:
Example 1
  • <!DOCTYPE html>
  • <html>
  • <body>

  • <h2>A defination List</h2>

  • <dl>
  • <dt>Coffee</dt>
  • <dd>- black hot drink</dd>
  • <dt>Milk</dt>
  • <dd>- white cold drink</dd>
  • </dl>

  • </body>
  • </html>

OUTPUT

A defination List

Coffee
- black hot drink
Milk
- white cold drink

NESTED LIST

  • <!DOCTYPE html>
  • <html>
  • <body>

  • <h2>Nested List</h2>

  • <ol>
  • <li>FRONTEND</li>
  • <ul>
  • <li>HTML</li>
  • <li>CSS</li>
  • <li>JAVASCRIPT</li>
  • <li>JQUERY</li>
  • <li>JASON</li>
  • <li>AJAX</li>
  • </ul>
  • <li>BACKEND</li>
  • <ul>
  • <li>JAVA</li>
  • <li>PYTHON</li>
  • <li>PHP</li>
  • <li>NODE JS</li>
  • <li>EXPRESS JS</li>
  • </ul>
  • <li>DATABASE</li>
  • <dl>
  • <dt>SQL</dt>
  • <dd>Structured Query Language</dd>
  • <dt>MYSQL</dt>
  • <dd>My Structured Query Language</dd>
  • <dt>MONGODB</dt>
  • <dd>No Sql Database</dd>
  • </dl>
  • </ol>
  • </body>
  • </html>

    • CSS

    • JQUERY
    • JASON
    • AJAX
